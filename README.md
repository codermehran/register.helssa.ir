# register.helssa.ir

سامانه ساده ثبت‌نام اولیه بیماران با Django. این پروژه یک فرم فارسی و راست‌به‌چپ برای دریافت نام، نام خانوادگی و شماره موبایل بیمار فراهم می‌کند و داده‌ها را در پایگاه داده ذخیره می‌کند.

## امکانات

- فرم ثبت‌نام فارسی با رابط راست‌به‌چپ
- اعتبارسنجی شماره موبایل ایران:
  - الزام به ۱۱ رقم
  - فقط عدد
  - شروع با `09`
  - جلوگیری از ثبت شماره تکراری
- نمایش خطاهای هر فیلد به زبان فارسی
- نمایش پیام موفقیت پس از ثبت اطلاعات
- جلوگیری از ارسال چندباره فرم با غیرفعال کردن دکمه ثبت پس از ارسال
- پنل مدیریت Django برای مشاهده و جست‌وجوی بیماران
- تست‌های واحد برای مدل، فرم، نما و قالب

## فناوری‌ها

- Python
- Django `5.2.14`
- SQLite برای محیط توسعه
- HTML/CSS/JavaScript ساده برای رابط کاربری

## پیش‌نیازها

- Python سازگار با Django 5.2
- `pip`
- محیط مجازی Python، پیشنهاد شده برای جلوگیری از تداخل وابستگی‌ها

## نصب و راه‌اندازی

ابتدا وارد پوشه پروژه شوید:

```powershell
cd C:\Users\Bartar\OneDrive\Apps\Desktop\register.helssa.ir
```

محیط مجازی بسازید و فعال کنید:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

وابستگی‌ها را نصب کنید:

```powershell
pip install -r requirements.txt
```

مایگریشن‌ها را اجرا کنید:

```powershell
python manage.py migrate
```

سرور توسعه را اجرا کنید:

```powershell
python manage.py runserver
```

سپس پروژه از این آدرس در دسترس است:

```text
http://127.0.0.1:8000/
```

## مسیرهای اصلی

| مسیر | توضیح |
| --- | --- |
| `/` | نمایش فرم ثبت‌نام بیمار |
| `/register/` | نمایش همان فرم ثبت‌نام |
| `/admin/` | پنل مدیریت Django |

## مدل داده

مدل اصلی پروژه `Patient` است:

| فیلد | نوع | توضیح |
| --- | --- | --- |
| `first_name` | `CharField(max_length=100)` | نام بیمار |
| `last_name` | `CharField(max_length=100)` | نام خانوادگی بیمار |
| `mobile` | `CharField(max_length=11, unique=True)` | شماره موبایل یکتا |
| `created_at` | `DateTimeField(auto_now_add=True)` | زمان ثبت رکورد |

## ساخت کاربر مدیر

برای ورود به پنل مدیریت، یک کاربر مدیر بسازید:

```powershell
python manage.py createsuperuser
```

بعد از اجرای سرور، وارد مسیر زیر شوید:

```text
http://127.0.0.1:8000/admin/
```

در پنل مدیریت می‌توانید بیماران ثبت‌شده را بر اساس نام، نام خانوادگی یا شماره موبایل جست‌وجو کنید.

## اجرای تست‌ها

برای اجرای تست‌های پروژه:

```powershell
python manage.py test
```

تست‌ها موارد زیر را پوشش می‌دهند:

- یکتا بودن شماره موبایل در مدل
- اعتبارسنجی فرم ثبت‌نام
- پیام‌های خطای فارسی
- ثبت موفق بیمار
- مدیریت خطاهای پایگاه داده
- محتوای قالب و وضعیت‌های رابط کاربری

## نکات توسعه

- فایل استایل اصلی در `patients/static/patients/css/style.css` قرار دارد.
- قالب فرم ثبت‌نام در `patients/templates/patients/register.html` قرار دارد.
- منطق ثبت اطلاعات در `patients/views.py` و اعتبارسنجی‌ها در `patients/forms.py` پیاده‌سازی شده‌اند.
- پس از تغییر مدل‌ها، این دستورات را اجرا کنید:

```powershell
python manage.py makemigrations
python manage.py migrate
```

## آماده‌سازی برای انتشار

تنظیمات فعلی برای توسعه محلی مناسب است. قبل از انتشار در محیط واقعی، این موارد باید تغییر کنند:

- مقدار `SECRET_KEY` را از فایل تنظیمات خارج کنید و از متغیر محیطی بخوانید.
- مقدار `DEBUG` را در محیط تولید `False` کنید.
- دامنه‌های مجاز را در `ALLOWED_HOSTS` تنظیم کنید.
- در صورت نیاز، SQLite را با پایگاه داده مناسب تولید مثل PostgreSQL جایگزین کنید.
- تنظیمات فایل‌های استاتیک، HTTPS، لاگ‌گیری و بکاپ پایگاه داده را کامل کنید.
- دسترسی به پنل مدیریت را محدود و ایمن کنید.
